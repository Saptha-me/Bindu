\# Local Setup for Bindu (Beginner Friendly)



\## Requirements

\- Python 3.12+

\- git

\- virtual environment support



\## 1. Clone \& Setup

```bash

git clone https://github.com/<your-username>/Bindu.git

cd Bindu

python -m venv .venv

\# Windows

.venv\\Scripts\\Activate.ps1

\# Mac/Linux

source .venv/bin/activate
